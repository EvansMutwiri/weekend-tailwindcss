What is a Static Website?

A static website consists only of static webpages. A static web page (sometimes called a flat page or a stationary page) is a web page that is delivered to the user's web browser exactly as stored, in contrast to dynamic web pages which are generated by a web application.

What are examples of Static Websites

Usually, static websites are a great beginning for more complicated project. Local businesses could host their menus, agencies could post service descriptions, freelancers could publish portfolios and resumes. There's also place for media like image galleries, videos and even image boards or web apps, if done correctly.
